# ToyProject_BulletinBoardSystem
JSP를 활용하여 간단한 게시판 웹페이지를 만들어보았습니다.

로그인과 회원가입, 글쓰기/수정/삭제 기능이 간단하게 구현되어 있습니다.

## 기술스택: JSP, bootstrap, MySQL, Apache Tomcat
